# agent-scope-diff v0.1.0

Initial release.

## Highlights

- Risk-focused diffs for agent manifests, MCP configs, and tool permission snapshots.
- JSON and YAML inputs.
- Human, JSON, Markdown, SARIF, and GitHub Actions output modes.
- Directory comparison with `--compare-dir`.
- Normalizer profiles for generic configs, OpenAI Agents-style manifests, Claude Desktop MCP configs, and simple LangGraph-style configs.
- Severity override rules and accepted finding baselines.
- CI support with `--fail-on-high`.
